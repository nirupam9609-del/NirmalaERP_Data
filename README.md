# Nirmala ERP Data

This repository contains ERP-related data and reports.

## 📂 Contents
- Database_Backups → JSON backup files
- Excel_Reports → Excel reports and data sheets
- PDF_Reports → Generated PDF reports
- nirmala_master_db.json → Main database file
- system_audit_trail.log → System logs

## ⚙️ Purpose
This project is used for:
- Data storage
- Backup management
- Report tracking

## 🚀 Usage
This repository can be used to:
- Maintain ERP data backups
- Track changes over time
- Store reports securely

## ⚠️ Note
Temporary files and system-generated files are ignored using `.gitignore`.
